This a employee management system using JDBC and Orcale database to
achieve the function such as add, delete, change and find for employee information.

EmpDao.java: System interface design
EmpDaoImpl.java: System interface implementation
Emp.java: Employee Class Design 
Test.calss: Test the function as add, delete, change and find
Util.class: tools for connection with the databasae
conn.properties: connection information
